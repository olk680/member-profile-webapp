# Member Profile Web Application

## Project Description
This project is a simple web application developed using PHP and SQLite. The main purpose of this application is to display and manage information about group members. Each member has a profile that includes basic details such as name, email, and an image.

## Features
- Display a list of group members
- Show individual member information
- Store data using SQLite database
- Simple and easy-to-use interface

## Technology Used
- PHP
- SQLite
- HTML/CSS

## System Architecture
This project follows a basic 3-tier architecture:
- Presentation Layer: HTML pages for user interaction
- Application Layer: PHP scripts handling logic
- Data Layer: SQLite database storing member information

## How to Run
1. Install PHP on your system
2. Place the project files in a folder
3. Open terminal and navigate to the project directory
4. Run a local PHP server:
   php -S localhost:8000

5. Open a browser and go to:
   http://localhost:8000
