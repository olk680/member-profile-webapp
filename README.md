# Member Profile Web App

This project is a simple web application built using PHP and MariaDB, deployed on a Raspberry Pi.

## Features
- Add, edit, view, and delete members (CRUD)
- Upload and display member images
- Store data using a relational database

## Technologies Used
- PHP
- MariaDB
- Apache Web Server
- Raspberry Pi

## How to Run
1. Start Apache and MariaDB
2. Place files in /var/www/html/member-profile-webapp
3. Open in browser:
   http://<pi-ip>/member-profile-webapp/

## Description
This project demonstrates a server-side PHP application with database integration running on a Raspberry Pi.
